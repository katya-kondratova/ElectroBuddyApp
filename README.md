# ElectroBuddyApp
A clean and functional iOS utility app built with SwiftUI for quick engineering calculations and unit conversions.

### Key Features:
* **Power Calculator:** Instant wattage calculation based on Amps and Volts. 
* **Safety Alert:** Features a visual warning (red text) when power exceeds 3500W to indicate high load.
* **Distance Converter:** Seamlessly converts Feet to Meters with robust input handling (supports both dot and comma separators).
* **Smooth UX:** Includes an animated Splash Screen with a bolt icon for a polished app launch experience.
* **Native UI:** Utilizes modern SwiftUI components like `TabView`, `NavigationStack`, and `Form`.

### Tech Stack:
* **Language:** Swift
* **Framework:** SwiftUI
* **Concurrency:** DispatchQueue for splash screen timing
* **Architecture:** Clean state management using `@State` and Computed Properties
